# function.parameters.properties是函数所需参数对象，对象里的key是参数名称，对象的值是参数类型和描述，
# 返回结果的arguments的参数需要根据函数列表中函数对应的function.parameters参数来进行提取，确保与function.parameters的参数名称一样。
例如函数列表中函数为：
{
    "function": {
        "name": "GetOrder",
        "description": "获取订单信息",
        "parameters": {
            "type": "object",
            "required": [],
            "properties": {
                "orded_id": {
                    "type": "string",
                    "description": "订单号"
                }
            }
        }
    },
    "type": "function"
}
返回应该为:
{
    "name": "GetOrder",
    "arguments": {"orded_id":"XXX"}
}
